These 20 components generate richly styled HTML via `NATIVE_PLSQL` region source ã CSP-compliant server-side rendering for Oracle APEX 24.2. Each tool creates a PL/SQL region that calls `sys.htp.p()` to emit self-contained HTML with embedded CSS. No CDN resources, no inline JavaScript libraries, no separate static file uploads required.

<Note>
  These components are rendered by APEX's `NATIVE_PLSQL` region source mechanism, which is CSP-compliant in APEX 24.2. All CSS is emitted via `sys.htp.p('<style>...</style>')` at render time, and JavaScript (where used) runs in the APEX page context without violating Content Security Policy. All user-supplied data passes through `APEX_ESCAPE.HTML()` before output.
</Note>

All tools share a common guard check: the page must already exist in the session (created with `apex_add_page()`, `apex_generate_crud()`, or similar). Every tool returns a JSON object with `status`, `region_id`, and `page_id`.

## Layout & Navigation

### `apex_add_hero_banner`

Add a full-width hero banner to a page. Renders a prominent welcome or header section with a gradient background, title, optional subtitle, inline KPI stats (with live SQL), and an optional call-to-action button.

<ParamField path="page_id" type="integer" required>Target page ID.</ParamField>
<ParamField path="title" type="string" required>
  Main banner title. Supports APEX substitution strings, e.g. `"Welcome, &APP_USER.!"`.
</ParamField>
<ParamField path="subtitle" type="string">Secondary text line under the title.</ParamField>
<ParamField path="stats" type="array">
  Up to 4 inline KPI items displayed inside the banner. Each object:
  ```json
  {"label": "Open Tickets", "sql": "SELECT COUNT(*) FROM TICKETS WHERE STATUS='OPEN'", "suffix": ""}
  ```
</ParamField>
<ParamField path="bg_color" type="string" default="unimed">
  Background color ã named (`unimed`, `blue`, `teal`) or hex (`#00995D`).
</ParamField>
<ParamField path="button_label" type="string">CTA button label (e.g., `"New Record"`).</ParamField>
<ParamField path="button_url" type="string">URL or `f?p=` reference for the CTA button.</ParamField>
<ParamField path="title_size" type="string" default="1.7rem">CSS `font-size` for the title.</ParamField>
<ParamField path="subtitle_size" type="string" default="1rem">CSS `font-size` for the subtitle.</ParamField>
<ParamField path="custom_css" type="string">Additional CSS rules injected into the style block.</ParamField>
<ParamField path="sequence" type="integer" default="5">Display order on page.</ParamField>

**Example:**

```python
apex_add_hero_banner(
    page_id=1,
    title="Welcome to HR Portal, &APP_USER.!",
    subtitle="Manage your workforce with ease.",
    bg_color="#1e88e5",
    stats=[
        {"label": "Employees",    "sql": "SELECT COUNT(*) FROM EMPLOYEES"},
        {"label": "Open Roles",   "sql": "SELECT COUNT(*) FROM JOB_POSTINGS WHERE STATUS='OPEN'"},
        {"label": "On Leave",     "sql": "SELECT COUNT(*) FROM LEAVE_REQUESTS WHERE STATUS='APPROVED' AND SYSDATE BETWEEN START_DATE AND END_DATE"},
    ],
    button_label="New Employee",
    button_url="f?p=&APP_ID.:11:&SESSION.::&DEBUG.:::",
)
```

### `apex_add_traffic_light`

  `AMBER` / `WARNING` / `EM_ANDAMENTO` ã amber
  `RED` / `ERROR` / `CANCELADA` ã red
</ParamField>

### `apex_add_activity_stream`

Add a scrollable audit/activity feed showing recent events in reverse-chronological order. Relative date labels are computed automatically: `"hûÀ minutos"`, `"3h atrûÀs"`, `"2d atrûÀs"`, or `DD/MM/YYYY`.

</ParamField>
<ParamField path="text_column" type="string" default="TEXT">Column for the activity description.</ParamField>
<ParamField path="date_column" type="string" default="DT">Column for the event timestamp (DATE or TIMESTAMP).</ParamField>
<ParamField path="icon_column" type="string">Column providing a Font Awesome icon class per row (optional).</ParamField>
<ParamField path="color_column" type="string">Column providing a hex color per row (optional).</ParamField>
<ParamField path="default_icon" type="string" default="fa-circle">Default icon class when `icon_column` is empty or not provided.</ParamField>
<ParamField path="default_color" type="string" default="blue">Default accent color.</ParamField>
<ParamField path="max_rows" type="integer" default="20">Maximum rows to display.</ParamField>

**Example:**

```python
apex_add_activity_stream(
    page_id=1,
    region_name="Recent Activity",
    sql_query="""
        SELECT DS_DESCRIPTION AS TEXT,
               DT_CREATED      AS DT,
               CASE DS_TYPE
                 WHEN 'CREATE' THEN 'fa-plus-circle'
                 WHEN 'UPDATE' THEN 'fa-edit'
                 WHEN 'DELETE' THEN 'fa-trash'
                 ELSE 'fa-circle'
               END AS ICON_COL
          FROM AUDIT_LOG
         ORDER BY DT_CREATED DESC
    """,
    icon_column="ICON_COL",
    max_rows=15,
)
```

### `apex_add_heatmap_grid`

Add a color-intensity heatmap matrix. Each cell's background opacity reflects its value relative to the maximum value in the dataset. SQL must return row labels, column labels, and numeric cell values.

<Note>
  SQL must return three columns for the row dimension, column dimension, and cell value. The tool dynamically collects unique row and column labels, then renders a full cross-tab grid. Empty intersections are rendered as blank cells.
</Note>

<ParamField path="page_id" type="integer" required>Target page ID.</ParamField>
<ParamField path="region_name" type="string" required>Region display name.</ParamField>
<ParamField path="sql_query" type="string" required>
  SQL returning row label, column label, and numeric value.
  ```sql
  SELECT TO_CHAR(ORDER_DATE,'DY') AS ROW_LABEL,
         TO_CHAR(ORDER_DATE,'HH24') AS COL_LABEL,
         COUNT(*) AS VALUE
    FROM ORDERS
   GROUP BY TO_CHAR(ORDER_DATE,'DY'), TO_CHAR(ORDER_DATE,'HH24')
   ORDER BY 1, 2
  ```
</ParamField>
<ParamField path="row_column" type="string" default="ROW_LABEL">Column for row labels.</ParamField>
<ParamField path="col_column" type="string" default="COL_LABEL">Column for column labels.</ParamField>
<ParamField path="value_column" type="string" default="VALUE">Column for numeric cell values.</ParamField>
<ParamField path="color" type="string" default="blue">Heatmap base color. High-intensity cells use this color; low-intensity cells use a tint.</ParamField>

**Example:**

```python
apex_add_heatmap_grid(
    page_id=5,
    region_name="Orders by Day & Hour",
    sql_query="""
        SELECT TO_CHAR(ORDER_DATE, 'DY')  AS ROW_LABEL,
               TO_CHAR(ORDER_DATE, 'HH24') AS COL_LABEL,
               COUNT(*) AS VALUE
          FROM ORDERS
         WHERE ORDER_DATE >= SYSDATE - 90
         GROUP BY TO_CHAR(ORDER_DATE, 'DY'), TO_CHAR(ORDER_DATE, 'HH24')
         ORDER BY 1, 2
```
